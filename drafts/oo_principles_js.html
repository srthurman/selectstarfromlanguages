Sun 05 April 2015

Javascript

So, the thing with Javascript is it doesn't have classes. Instead, you create functions which serve as the blueprints for making objects. These functions are your defacto classes. It may seem odd that you would use a function to create objects, but in Javascript, functions are (takes off the Scooby Doo villain mask)...objects! Functions are objects, and you can use them to create other objects. Confused yet?
